# QtTicker
Ticker software using Qt.
DirectX 11 is used for the drawing process, enabling smooth rendering using the GPU.

## Dependency
- MSVC 15+
- Qt 5.13.1
- DirectX 11.1

## Setup
Before running, you need to configure Visual Studio to use Qt.
1. git clone this repository.
2. open 'QtTicker.sln'
3. Please Build & Run!!
